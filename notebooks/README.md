# Jupyter Notebooks

Tento adresár obsahuje Jupyter notebooks pre exploratívnu analýzu dát (EDA) a experimentovanie.

## Notebooks

- `01_eda.ipynb` - Exploratívna analýza Kaggle House Prices dataset
- `02_feature_engineering.ipynb` - Feature engineering experimenty
- `03_model_training.ipynb` - Lokálne trénovanie modelov

## Použitie

```bash
# Spusti Jupyter v SageMaker Studio alebo lokálne
jupyter lab
```

## SageMaker Studio

Tieto notebooks sú navrhnuté pre SageMaker Studio, kde môžeš:
- Použiť Data Wrangler pre feature engineering
- Trackovať experimenty cez MLflow
- Spúšťať SageMaker Training Jobs priamo z notebookov
